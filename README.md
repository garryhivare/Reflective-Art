# Reflective-Art

**ReflectiveArt - Photography Showcase**
ReflectiveArt is a web-based gallery platform designed to showcase the work of world-class artists, photographers, and sculptors. This specific module focuses on a curated list of top professional photographers, providing a clean, responsive interface for users to explore their portfolios.

Responsive Navigation: Utilizes a Bootstrap 5 fixed-top navbar with an offcanvas mobile menu for seamless browsing across devices.

Artist Profiles: A dedicated section showcasing top photographers using high-quality card layouts, rounded thumbnails, and shadow effects.

Dynamic Links: Each photographer card includes a "Learn More" call-to-action that redirects to their professional portfolios or interviews.

Professional Footer: A multi-column footer including company information, community links, a newsletter/search input, and social media integration.
